# Image Gallery Admin Dashboard

React-based admin dashboard for managing images and users.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Create `.env` file:
```
REACT_APP_API_BASE_URL=http://localhost:5000/api
```

3. Start the development server:
```bash
npm start
```

The dashboard will be available at `http://localhost:3000`

## Features

- Admin login/register with email and password
- Upload images with title and description
- Edit image metadata
- Delete images
- View all uploaded images
- Responsive design
